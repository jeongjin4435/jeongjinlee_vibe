# PERSONA LAB - 전문 MBTI 성향 분석

## 🌊 Ocean Blue Premium Edition (V2)

### 1. 개요
기존 MBTI 검사 프로그램을 'Ocean Blue' 디자인 시스템과 전문적인 분석 기능을 갖춘 프리미엄 버전으로 대대적으로 업그레이드했습니다.

### 2. 주요 업데이트 사항

#### 💎 Ocean Blue UI 디자인 시스템
- **모던한 컬러 팔레트**: Ocean Blue (#0ea5e9)를 메인으로 한 청량하고 신뢰감 있는 디자인.
- **아티클 레이아웃**: 가독성 최적화를 위한 768px 중앙 컨테이너 레이아웃 적용.
- **인터랙션 강화**: 모든 버튼과 선택지에 Scale Transform (1.02배) 및 부드러운 애니메이션 적용.
- **Rounded Soft 스타일**: 모든 컴포넌트에 12px 곡률 적용으로 현대적이고 친근한 느낌 구현.
- **다크 모드 지원**: 사용자 환경에 맞는 라이트/다크 테마 전환 기능.

#### 🚀 전문 분석 기능 (V2)
- **SVG 레이더 차트**: 4가지 성향 지표를 한눈에 볼 수 있는 시각화 차트 도입.
- **유형별 궁합 정보**: 16가지 유형별 '환상의 궁합' 및 '주의할 궁합' 데이터 추가.
- **진로 및 유명인**: 유형별 추천 직업군과 대표 유명인 데이터 보강.
- **문항 네비게이터**: 64문항 전체 진행 상태 확인 및 즉시 이동 기능.
- **UX 편의성**: 답변 선택 시 자동 다음 문항 이동 옵션 추가.

### 3. 기술 스택
- HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS)
- Google Fonts (Inter, Pretendard)
- LocalStorage 기반 데이터 저장
